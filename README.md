# LearnPark NEET Academy Website

Multi-page React website for LearnPark NEET Academy, built with Vite, Tailwind CSS, and Framer Motion.

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 3
- Framer Motion
- React Router DOM

## Local Development

1. Install dependencies:

	npm install

2. Start dev server:

	npm run dev

3. Build for production:

	npm run build

## Deployment

### Vercel

1. Import this GitHub repository in Vercel.
2. Framework preset: `Vite`.
3. Build command: `npm run build`.
4. Output directory: `dist`.
5. Deploy.

### Netlify

1. Import this GitHub repository in Netlify.
2. Build command: `npm run build`.
3. Publish directory: `dist`.
4. Deploy.

## Notes

- Main app code is under `src/`.
- Global layout/navigation is in `src/components/SiteLayout.jsx`.
- Home storytelling section is in `src/pages/HomePage.jsx`.
